The zip file has the following directory structure

\project_main\npsproject
						\npsproject
						\eventsapp


After extracting the zip file, navigate to \eventsapp directory under \project_main\npsproject.

Here, 

models.py 	- contains the table structures and the relationships
urls.py		- contains the url structure or templates and the name of corresponding view names
views.py	- contains the code for the views referred in urls.py
serializers.py - contains the information about how to serialize deserialize the data
